Schedule Viewer — Інструкція користувача

Повний гайд по всіх сторінках і кнопках · Оновлено: 30.04.2026
PDF: натисніть Ctrl+P → Save as PDF → увімкніть Background graphics.

1. Швидкий старт

  1. Відкрийте index.html (desktop) або index2.html (mobile).
  2. Оберіть режим: Група / Викладач / Ауд.
  3. Оберіть факультет, форму, курс, групу або викладача.
  4. Вкажіть дати та натисніть + Додати.

2. Основні сторінки

2.1 Головна (Desktop/Mobile)

  • Пошук і порівняння розкладу по групах/викладачах.
  • Фільтри: офлайн/онлайн, дисципліни, типи занять.
  • Експорт, календар, обране, тема, очистка даних.

2.2 Сесія — session.html

  • Фільтри по сесії, формі, групі, викладачу, дисципліні, даті, типу контролю.
  • Експорт відфільтрованої таблиці в XLSX.

2.3 Адмін сесій — session-admin.html

  • Завантаження DOCX, парсинг, відправка в API.
  • Операції з сесіями: перейменування, архів, видалення, backup.

2.4 Конструктор сесії — session-constructor.html

  • Ручне редагування предметів/груп/викладачів/дат/часу/аудиторій.
  • Валідація і конфлікти в реальному часі.

2.5 Підготовка до сесії — session-prep.html

  • Таблиця Предмет | Група | Викладачі для обраних курсів.
  • Пошук, фільтрація, копіювання, експорт в Excel.

2.6 Smart Day — smart.html

  • Рейтинг зручності, теплокарта, what-if симулятор, live board.

2.7 Week Builder — builder.html

  • Збір тижня, аналіз конфліктів/дублів/вікон, оптимізація, Excel.

2.8 Course Day — course-live.html

Автозавантаження вимкнено: спочатку оберіть фільтри, потім натисніть Оновити зараз.
  • Heatmap, конфлікти аудиторій, навантаження/вікна викладачів.
  • Порівняння з датою, тільки зміни, пресети, експорт, журнал дій.

3. Типові задачі

  • Розклад групи: Головна → Група → фільтри → + Додати.
  • Розклад викладача: Головна → Викладач → фільтри → + Додати.
  • Вільні аудиторії: Головна → Ауд. → дата → Скан.
  • Завантажити DOCX сесії: Session Admin → пароль → файли → парсинг → API.
  • Підготовка “предмет-група-викладачі”: Session Prep → вибір курсів → сформувати.

4. Коли щось не так

  • Оновіть сторінку через Ctrl+F5.
  • Перевірте дату та активні фільтри.
  • Для адмін-дій перевірте пароль.
  • У Course Day обов'язково натискайте Оновити зараз.
